# 🤖 NLP Chatbot using Transformers

This project is a simple chatbot built using Hugging Face Transformers. It uses a pre-trained DialoGPT model to generate human-like responses.

## 📌 Technologies Used
- Python
- Hugging Face Transformers
- PyTorch

## ⚙️ Features
- Interactive chatbot
- Dynamic response generation
- Maintains conversation history

## 🔄 Workflow
User Input → Model → Response → Output

## 🎯 Conclusion
This project helped me understand how transformer models are used in real-world NLP applications.
